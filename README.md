# Gimmi Burga
## A burger/food delivery website

This project runs on both a front and back end server.

**Instructions:** install `npm install` then run `npm start` from console <- both front and back end.

### Front End Stack:

* Font Awesome
* React
* React Dom
* React Helmet
* React Spring
* Styled Components
* Redux w/ Reselect
* Redux Persist

### Back End Stack

* Express
* Mongoose
* Cors
* DotEnv
